# IT Portfolio - Active Directory & Cloud Identity Labs
This project documents a hands-on Active Directory lab built to practice common IT Support/System Administration tasks. This lab focuses on user and group management, access control through security groups, and troubleshooting authentication/permission issues in a Windows domain environment.

## Projects

### Active Directory User Management Lab
- Built a Windows Server domain environment
- Administered users, groups, and OUs
- Performed password resets and access troubleshooting

Folder: `active-directory-lab`

---

### Azure Identity & Access Management
- Created and managed cloud users and groups
- Implemented MFA and Conditional Access
- Reviewed sign-in logs for troubleshooting
- Managed Microsoft 365 licenses

  Folder: `azure-iam-lab`

  ## Skills Demonstrated
  - Active Directory (AD DS)
  - Microsoft Entra ID (Azure AD)
  - Identity & Access Management (IAM)
  - MFA & Conditional Access
  - IT Support & Troubleshooting
